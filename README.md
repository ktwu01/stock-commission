## 简介

一个单页计算器，方便估算买卖 A 股时的佣金、印花税、过户费和目标净回款。通过输入股票代码、手数、费率与目标收益，即可得到：

- 买入总成本(发生金额)与综合费率
- 卖出目标价或自定义卖价下的费用明细与综合费率
- 卖出净回款、资本收益、以及必要的“所需卖出价”提示
